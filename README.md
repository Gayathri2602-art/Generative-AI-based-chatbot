RAG Chatbot (Llama 2)

A Retrieval-Augmented Generation (RAG) chatbot that retrieves relevant information from documents and generates accurate, context-aware responses using Llama 2.

----

🚀 Overview

This project builds a document-based AI assistant:

Documents are embedded and stored in Pinecone

Relevant content is retrieved based on user queries

Context is passed to Llama 2

The chatbot generates grounded responses

---

🛠️ Tech Stack

Python

LangChain

Flask

Llama 2

Pinecone

Sentence Transformers

---

📌 Features

🔍 Semantic search with Pinecone

🤖 Context-aware responses

📚 Document-based Q&A

🌐 Flask web interface


----

How It Works

User Query → Embeddings → Pinecone Retrieval → Llama 2 → Response
